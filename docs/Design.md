# Prashworld — Design System

> **Version:** 1.0 | **Last Updated:** 2026-06-13

## Philosophy
Organic, earthy, content-first. Nature-inspired warmth, not cold AI templates. Benchmarked against Linear, Notion, Airbnb, Stripe.

## Colors
- **Primary:** Deep Forest green (`#2d8a3e` → scale 50-900)
- **Accent:** Golden Hour gold (`#d4940d` → scale)
- **Warm:** Terra tones (`#c26a3d` → scale)
- **Neutrals:** Warm gray (`#faf9f7` bg → `#1a1714` text)
- **Semantic:** Success green, Warning gold, Error red, Like red `#e04848`
- **Dark Mode:** Deep earth `#0f0e0c` bg, `#ebe7e0` text

## Typography
- **Font:** Outfit (Google Fonts) — clean, modern
- **Scale:** xs(11px) → 3xl(40px), weights 400-800

## Spacing: 4px base (space-1 to space-20)
## Radius: sm(6px) → full(9999px)
## Shadows: Warm-tinted (xs → 2xl) + glow variants for focus
## Buttons: Primary, Secondary, Ghost, Danger, Google — with hover/active/focus states
## Inputs: Inner shadow, focus glow, error state
## Avatars: xs(24px) → 2xl(128px) with placeholder gradient

## Micro-Interactions
- Heart pop animation, page fadeInUp, skeleton shimmer, button press scale, toast slide
- Stagger children for lists

## Responsive: Mobile-first, bottom nav → sidebar at 1024px
## Dark Mode: CSS custom properties + `data-theme="dark"` + system preference detection
